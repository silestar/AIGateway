# Changelog

## [0.1.0] - 2026-05-04
### Added
- 完成全部核心设计文档（架构设计、模块详细设计、WebUI原型）
- 确定项目名为 AIGateway（AGW）
- 确定项目目录结构规范
- 完成分阶段实施策略（7个阶段）
- 新建集中式管理 API 接口规范文档

### Changed
- 路由流程增加消费者配额检查步骤（429 前置拦截）
- `account_select` 插件钩子改为预过滤语义，移除强制指定账号能力
- `realtime` 端点明确以 Redis 计数器为准，聚合表仅用于历史查询
- 聚合调度频率从每小时改为每 5 分钟
- 密钥脱敏规则统一为 `sk-...****`，不暴露真实字符
- 账号池增加 `max_probe_recover_per_cycle` 配置项，支持每周期恢复多个账号
- 新增全局健康巡检机制，与按需探测互补
- 新增阶梯式冷却机制（一级2小时 / 二级24小时）
