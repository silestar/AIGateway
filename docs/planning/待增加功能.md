# AIGateway 待增加功能

## 上游定时自动检查模型变动

- **优先级**：中
- **描述**：定时自动调用上游 API 获取当前支持的模型列表，与本地已配置的模型进行比对，标记出上游已下线的模型，并通知管理员处理
- **当前状态**：仅支持手动触发"上游更新"功能（渠道列表页 → 更多操作 → 上游更新）
- **预期实现**：
  - 在系统配置中增加"上游模型检查间隔"参数（如 24h）
  - 定时检查所有渠道的上游模型变动
  - 发现下线模型时通过系统通知或日志告警
  - 可选：自动禁用已下线模型（需管理员确认模式）
