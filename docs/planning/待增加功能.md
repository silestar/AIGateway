# AIGateway 待增加功能

## 上游定时自动检查模型变动

- **优先级**：中
- **描述**：定时自动调用上游 API 获取当前支持的模型列表，与本地已配置的模型进行比对，标记出上游已下线的模型，并通知管理员处理
- **当前状态**：仅支持手动触发"上游更新"功能（渠道列表页 → 更多操作 → 上游更新）
- **预期实现**：
  - 在系统配置中增加"上游模型检查间隔"参数（如 24h）
  - 定时检查所有渠道的上游模型变动
  - 发现下线模型时通过系统通知或日志告警
  - 可选：自动禁用已下线模型（需管理员确认模式）

## 插件市场功能

- **优先级**：低（第二阶段）
- **描述**：对接外部插件市场，实现插件的在线浏览、一键安装、自动更新
- **当前状态**：
  - API 已预留：`GET /api/plugins/market`（代理市场列表）、`POST /api/plugins/market/:name/install`（一键安装）
  - 前端已预留：市场按钮（当前置灰并提示"市场服务未配置"）
  - 配置项已预留：`plugin.marketplace_url`（市场 API 地址）、`plugin.marketplace_token`（认证 Token）
- **预期实现**：
  - 独立部署的插件市场管理端（独立仓库）
  - AGW 通过 `plugin.marketplace_url` 配置对接
  - 前端市场 Tab：展示可用插件列表、版本信息、安装按钮
  - 安装流程：从市场下载 ZIP → 调用本地安装流程（复用 `/api/plugins/upload` 逻辑）
  - 版本检查：定时检查已安装插件是否有新版本，提示更新
  - 市场认证：通过 `plugin.marketplace_token` 鉴权
